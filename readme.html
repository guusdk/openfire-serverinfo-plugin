ServerInfo Plugin Readme

Overview

The ServerInfo plugin exposes basic information regarding the XMPP domain, including incoming and outgoing s2s connections and user counts. It is modeled after Thomas Leister's 'mod_serverinfo' for Prosody.

Installation

Copy serverinfo.jar into the plugins directory of your Openfire installation. The plugin will then be automatically deployed. To upgrade to a new version, copy the new candy.jar file over the existing file.

Using the Plugin

When deployed, the json information will be served on the same ports as are the Admin Console, typically 9090 and 9091. The path will be /plugins/serverinfo/:

  • http://<your-server-name>:9090/plugins/serverinfo/ (unencrypted HTTP)
  • https://<your-server-name>:9091/plugins/serverinfo/ (encrypted HTTPS)
